Snake Xenzia

Snake Xenzia is a classic snake game implemented in Python using the Turtle graphics library. It brings the nostalgic joy of the iconic mobile game right to your computer. In this game, players control a snake that grows longer as it consumes food while avoiding collisions with the walls and its own tail.
